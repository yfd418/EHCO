# 📂 文档一：Echo 产品需求文档 (PRD)

**版本：** v1.0 (MVP)
**核心理念：** *Less noise, more signal.* (回归纯粹沟通，拒绝社交噪音)

## 1. 产品定义

Echo 是一个运行在 Web 端的极简即时通讯工具。它服务于厌倦了主流社交软件臃肿功能、追求界面美感与沟通效率的用户。它没有朋友圈，没有算法推荐，只有你和你想交谈的人。

## 2. 核心功能清单 (MVP)

### P0 (必须有 - 第一周目标)

1. **极简身份 (Identity):**
* 支持邮箱/密码注册，或魔法链接 (Magic Link) 登录。
* 每个用户拥有唯一的 `username` (用于搜索) 和 `Display Name` (昵称)。
* 用户状态显示：在线/离线 (Online/Offline)。


2. **社交连接 (Connections):**
* **搜索添加：** 只能通过精准搜索 `username` 添加好友。
* **好友列表：** 按“最近活跃”排序。


3. **核心会话 (Conversation):**
* **即时通讯：** 消息必须在 <100ms 内同步 (Real-time)。
* **消息类型：** 纯文本 (支持 Markdown 渲染)、Emoji、单张图片。
* **状态反馈：** 发送中 -> 已送达。



### P1 (可以稍后 - 第二周目标)

1. **未读计数：** 红色小圆点提示未读消息数。
2. **对方正在输入：** 聊天框顶部显示 "Typing..."。
3. **移动端适配：** 手机浏览器访问时自动切换为单栏布局。

## 3. UI/UX 设计规范 (The "Echo" Look)

* **色彩体系 (Palette):**
* **背景:** `#FFFFFF` (纯白) / `#F9FAFB` (极淡灰)
* **文字:** `#111827` (深灰几近黑)
* **强调色:** `#000000` (纯黑 - 用于按钮/我的消息气泡)
* **辅助色:** `#E5E7EB` (边框灰)


* **排版 (Typography):**
* **Logo/标题:** Serif 衬线体 (如 Playfair Display / Noto Serif SC) —— *体现质感*。
* **正文:** Sans-serif 无衬线体 (Inter / Noto Sans SC) —— *确保易读*。


* **交互 (Interaction):**
* **无刷新:** 所有操作必须是 SPA (单页应用) 体验。
* **直角美学:** 消息气泡采用部分直角设计 (如 Demo 所示)，区别于传统的全圆角。
